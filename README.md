# Dino_Hunter_Game
  First Person Shooter Survival Game Built Using Unity 
  
# Features 
 1. Design Patterns Used 
  
     i) Model View Controller
 
     ii) Observer Pattern 

     iii) State Machine

     iv) Object Pooling 

     v) Singleton
     
2. Scriptable Objects.

3. Player Inventory System.

4. Dynamic Achievement System.

4. Occlusion Culling To Optimized Performance.
 
4. Reactive Enemies
